This report is generated by the test data. There is a standard value for each bio measurement, and the standard value is the median of the lowest DOSE results. In this report, the normal bio range is set to its standard value Â± 25%.

For example:
Chemical Name: Isoeugenol
The lowest DOES: 32.8 mg/kg
ALT Standard value: 61.5 g/dL
Normal ALT range: between 46.125 g/dL (61.5X75%) and 76.875 g/dL (61.5X125%)

Ab Count - Number of abnormal records | Ab Rate - The proportion of abnormal records
